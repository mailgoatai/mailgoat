# Example: E-commerce Checkout

Next.js-style ecommerce workflow showing order confirmation, shipping update, and receipt emails.

## Features

- Order placed -> Confirmation
- Shipping -> Tracking update
- Delivered -> Receipt + review request

## Setup

1. `cp .env.example .env`
2. `npm install`
3. `npm test`
4. `npm run dev`

## Templates

- `emails/order-confirmation.html`
- `emails/shipping-update.html`
- `emails/receipt.html`
