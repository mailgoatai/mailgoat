# MailGoat Community

MailGoat is built for developers and operators running agentic email systems.

## Where To Get Help

- GitHub Discussions (`Q&A`): usage and troubleshooting questions
- GitHub Issues: reproducible bugs and actionable feature work
- Security reports: see `SECURITY.md`

## How To Participate

- Share ideas in Discussions (`Ideas`)
- Share projects in Discussions (`Show and Tell`)
- Read `CONTRIBUTING.md` before opening pull requests

## Response Expectations

- Community Q&A: best effort
- Bug triage: usually within a few business days
- Security reports: acknowledged within 72 hours

## Community Norms

Be respectful, concise, and specific. Follow `CODE_OF_CONDUCT.md`.
