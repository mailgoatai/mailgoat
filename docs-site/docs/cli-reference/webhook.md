Webhook server and Postal webhook management

## Usage

```bash
mailgoat webhook --help
```

## Arguments

- Webhook id
- Event id to replay

## Common Options

- Bind address
- Port to listen on
- Webhook path
- Webhook HMAC secret (or MAILGOAT_WEBHOOK_SECRET)
- Path to SQLite inbox database
- Path to webhook event log file
- Run in detached background process
- PID file path
- Enable HTTPS/TLS
- TLS certificate path
- TLS private key path
- Rate-limit window in ms
- Rate-limit max requests per IP per window
- Start ngrok tunnel and print URL
- Webhook name
- Follow logs continuously
- Path to log file
- Path to SQLite inbox database
- Path to webhook event log file
