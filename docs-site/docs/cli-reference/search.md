title: search
---

Search messages from local inbox cache with filters

## Usage

```bash
mailgoat search --help
```

## Common Options

- Search messages from specific sender
- Search messages to specific recipient
- Search messages by subject (partial match)
- Search message body content
- Messages after date (ISO format or relative like 7d)
- Messages before date (ISO format or relative like 7d)
- Search messages with specific tag
- Only messages with attachments
- Maximum number of results (default: 25)
- Sort by: date, from, to, subject (default: date)
- Sort order: asc, desc (default: desc)
- Path to inbox cache JSON file
- Output results as JSON
