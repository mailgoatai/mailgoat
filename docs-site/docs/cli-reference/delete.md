Delete email messages

## Usage

```bash
mailgoat delete --help
```

## Arguments

- Message ID to delete (for single deletion)

## Common Options

- Delete messages older than duration (e.g., 30d, 2w, 6h)
- Delete messages from specific sender
- Delete messages to specific recipient
- Delete messages with specific tag
- Delete messages matching subject (partial match)
- Show what would be deleted without actually deleting
- Maximum number of messages to delete (for bulk operations)
- Output result as JSON
