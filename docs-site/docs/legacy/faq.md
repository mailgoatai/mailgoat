# FAQ

## What is MailGoat?

MailGoat is a CLI-first email toolkit for AI agents and automation workflows, designed around Postal.

## Do I need Postal?

Yes. MailGoat uses Postal APIs and data sources.

## Is there an admin UI?

Yes. Run `mailgoat admin serve` and open `/admin`.

## Where should I ask questions?

Use GitHub Discussions in the `Q&A` category.

## How do I report bugs?

Open a GitHub Issue with reproduction steps and expected vs actual behavior.

## Can I request features?

Yes, use Discussions (`Ideas`) first, then convert to an issue when scoped.
